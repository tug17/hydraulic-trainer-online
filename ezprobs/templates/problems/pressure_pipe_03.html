{% extends "problem.html" %} {% block title %}Betrieb einer Pumpturbine{% endblock %} {% block description %} Der Transport einer Flüssigkeit von Punkt A nach Punkt B ist energetisch immer verlustbehaftet.
Nach Abzug der gesamten Verluste vom Höhenunterschied der Behälter erhält man die \(Nutzfallhöhe\) \(h_{N}\).
Diese steht nun zur Energiegewinnung mit Hilfe einer Turbine bereit.

Die \(manometrische\ Förderhöhe\) \(h_{m}\) hingegen ist die Höhe die inklusive Höhenunterschied und Verlusten
mit Hilfe einer Pumpe überwunden werden muss um einen Durchfluss zu gewährleisten.

In diesem Beispiel wird zwischen 3 Zuständen unterschieden. Das Wasser fließt:
1. von dem niedrigeren zu dem höheren Behälter. Hierfür wird eine Pumpleistung benötigt.
2. von dem höheren zu dem niedrigeren Behälter mit gemäßigtem Durchfluss. Hierbei kann eine Pumpe betrieben werden.
3. von dem höheren zu dem niedrigeren Behälter mit einem erhöhten Durchfluss. Hierfür wird eine Pumpleistung benötigt.

Ziel dieses Beispiels ist es, die benötigte Pump-, bzw. erbrachte Turbinenleistung zu berechnen.
Bei der Eingabe der Werte ist zu beachten, dass sich positive Werte auf eine Durchflussrichtung von links nach rechts beziehen
und negative Werte auf eine Durchflussrichtung von rechts nach links.
